Github + Gridea 搭建个人博客

Gridea 简介：

官网：https://gridea.dev/ 🚀
Github 主页：https://github.com/getgridea/gridea 🚀

  Gridea 是一款开源的博客平台，自带后台客户端，提供撰写、托管、自定义域名的全功能 Blog 工具 🞨 💡 轻量的思考箱。

  

Github + Gridea 配置流程

  

【1】注册 Github 账号

  首先，我们在 Github 官网注册一个 Github 账号，建议使用谷歌邮箱注册，注册完毕后登录进入主页面。

【2】创建 github.io 仓库

  点击主页面右上角 "+" ，然后点击 New Repository 创建一个新仓库。

  我们需要创建一个与自己 Github 账号名称相同的特殊仓库。例如，小胖君的 Github 账号名称是 karley7，所以小胖君创建的仓库名称为 karley7.github.io，其中 .github.io 为固定格式，不能省略。小胖君已经创建过，因此提示该仓库已存在。
  我们也可以为仓库添加一些描述信息，同时创建一个介绍仓库的 Markdown文件，最后我们点击右下角生成按钮即可创建成功。

  点击主页面右上角的头像，然后点击 Your Profile即可查看刚才创建成功的仓库。

  

【3】创建仓库 Token

  点击主页面右上角的头像，然后点击 Setting进入设置页面，点击左侧导航栏最下端Developer settings。

  进入 Developer settings页面后，在左侧导航栏选择Personal access tokens里面的Tokens (classic)，点击右上角的Generate new token，选择Generate new token (classic)。

  进入Token生成页面，选择 Token 有效时间，勾选 Token 的访问权限，最后点击页面最下方的生成按钮即可生成 Token。
  生成后，请及时复制 Token ，因为它只显示一次，我们后面需要在 Gridea 中输入 Token。

  

【4】下载 Gridea 开源客户端

  我们在 Gridea 官网下载开源客户端，安装后打开客户端。在左侧导航栏选择 远程，在弹出的页面依次填入信息后，点击保存。最后点击左下角检测远程连接看是否连接成功。

  连接完毕后，我们的个人博客就搭建完成了，点击左侧导航栏的同步，同步成功后，我们就可以通过https://用户名.github.io来访问我们的个人博客网站，例如小胖君的个人博客网址是：https://karley7.github.io/。

【5】更换 Gridea 主题

  点击 Gridea Theme 进入 Gridea 的主题市场，选择自己喜欢的主题，然后点击下载压缩包。小胖君用的是 porky。

  将压缩包解压到 Gridea 的本地目录文件的 theme文件夹下面，Gridea 的本地目录文件可以点击 Gridea 客户端页面左下角的编辑按钮进行查看。

  关闭客户端，再次进入就可以在主题栏里选择自定义主题啦，点击同步后，就可以在个人博客网站自动同步该主题。给大家看看小胖君的个人博客主页面吧！

  

小结

  Gridea 官放也提供了搭建个人博客的教程，但是较为粗略，因此小胖君做了这期博客，希望对大家有所帮助，我们下期再见！
